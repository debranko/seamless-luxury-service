Removable Module

Dual Communication

RF radio + WiFi. Voice-to-text uses WiFi. Everything else runs on RF. If one drops, the other takes over. Two independent channels, switching automatically.

Multi-Frequency RF

Supports 433, 868, and 915 MHz. The system detects the vessel's location and switches to the correct regional frequency automatically. Install once, works everywhere.

Server Failover

Primary server down? Backup takes over with full database replication. No backup? Repeaters form an emergency mesh network. Service continues.

OTA Updates

Buttons, repeaters, server - all updated remotely. No truck rolls, no pulling the yacht out of charter. Push from shore, every device receives it.

Power Independent

Buttons run nine months. Repeaters have backup batteries. If the yacht loses all power, buttons and repeaters continue on RF. Guest can still call.

End-to-End Ack

Every call gets a unique ID. Button waits for server confirmation. Server confirms delivery to crew, confirms who accepted, sends feedback to guest. Full chain, no gaps.

So I built the system I wished existed. Not from a boardroom - from the crew mess, from the engine room, from years of doing the work.
